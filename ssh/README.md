# ApexOS App: Terminal & SSH

Allow logging in remotely to ApexOS using SSH or just the web terminal with Ingress.

![Supports aarch64 Architecture][aarch64-shield] ![Supports amd64 Architecture][amd64-shield]

## About

Setting up an SSH server allows access to your ApexOS folders with any SSH
client. It also includes a command-line tool (`apex`) to access the ApexOS API.
