# Apex Connect+ Add-on: CEC Scanner

## Installation

Follow these steps to get the add-on installed on your system:

1. Navigate in your Apex Connect+ frontend to **Settings** -> **Add-ons** -> **Add-on store**.
2. Find the "CEC Scanner" add-on and click it.
3. Click on the "INSTALL" button.

## How to use

This add-on has no configuration and runs out of the box.

1. Start the add-on.
2. Check the add-on log output to see the result.
